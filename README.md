# Global Environmental Infrastructure Dashboard

This Power BI dashboard analyzes global environmental and infrastructure indicators such as CO2 emissions, electricity access, internet usage, and fossil fuel subsidies.

## Project Overview

The goal of this project is to visualize environmental trends and infrastructure development across different countries.

## Dataset

The dataset contains:

- Country
- Year
- CO2 emissions (metric tons per capita)
- Electricity access (%)
- Internet users (%)
- Fossil fuel subsidies (% of GDP)

## Tools Used

- Power BI
- Data Visualization
- Data Analysis

## Key Insights

- Developed countries show higher internet penetration.
- CO2 emissions are declining in some developed economies.
- Electricity access is improving in developing countries.
